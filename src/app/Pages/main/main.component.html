DJI Mavic PRO

Данное приложение представляет собой интерактивную инструкцию для квадрокоптера DJI Mavic PRO. В ней вы сможете изучить квадрокоптер подробнее, узнать, как правильно ремонтировать и обслуживать устройство.

Использование приложения

В данном приложении доступно изучение квадрокоптера посредством просмотра анимаций сборки и разборки квадрокоптера. Чтобы открыть выпадающее меню, содержащее элементы навигации, нужно нажать на соответствующую кнопку:

Для воспроизведения анимации необходимо открыть сцену. Для этого нужно нажать на одну из кнопок для запуска выбранного набора сценариев. Выберите один из наборов и нажмите на одну из кнопок:
